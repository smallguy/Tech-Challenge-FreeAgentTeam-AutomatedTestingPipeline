2. 生成≥10个前端+≥10个后端测试代码（覆盖页面链接/控件交互/流程数据/业务逻辑/接口协议/数据存储/压力测试，1:1匹配技能要求）；

6. 输出自动化测试报告（含通过用例概况、失败用例详情、失败描述、通过率、测试覆盖率、修复建议），并保存在文件中。
